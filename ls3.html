GIỚI THIỆU VỀ CÂU LỆNH ĐIỀU KIỆN

Định nghĩa:

Câu lệnh điều kiện là cấu trúc trong JavaScript cho phép chương trình kiểm tra một điều kiện logic, và thực thi các đoạn mã khác nhau tùy theo điều kiện đó là đúng (true) hay sai (false). Nói cách khác, nó cho phép chương trình "ra quyết định".

Mục tiêu học tập:

  • Hiểu khái niệm và tác dụng của câu lệnh điều kiện
  • Sử dụng thành thạo các loại câu lệnh if, if...else, if...else if...else
  • Áp dụng toán tử ba ngôi (ternary) trong thực tế
  • Sử dụng câu lệnh switch...case hiệu quả
  • Kết hợp toán tử logic &&, ||, ! trong điều kiện
  • Giải quyết các bài toán thực tế bằng câu lệnh điều kiện

CÁC KHÁI NIỆM CƠ BẢN

Mục đích của câu lệnh điều kiện

Câu lệnh điều kiện giúp chương trình xử lý linh hoạt tùy theo tình huống cụ thể, cho phép kiểm tra dữ liệu, trạng thái và thực hiện hành động phù hợp.

Là nền tảng để viết logic cho các tính năng:

  • Kiểm tra đăng nhập: Xác thực thông tin người dùng
  • Tính điểm và xếp loại: Đánh giá kết quả học tập
  • Kiểm tra quyền truy cập: Phân quyền người dùng
  • Hiển thị thông báo: Thông báo lỗi, cảnh báo
  • Xử lý giao dịch: Kiểm tra số dư, điều kiện thanh toán

Tác dụng của câu lệnh điều kiện

Tác dụng Giải thích Ví dụ ứng dụng
Rẽ nhánh chương trình Dựa vào kết quả điều kiện, chương trình chạy theo nhánh khác nhau Chọn đường đi trong game, hiển thị giao diện khác nhau
Kiểm tra dữ liệu Xác định giá trị có hợp lệ, có tồn tại không Validate form, kiểm tra định dạng email
Tăng tính thông minh Cho phép phần mềm phản ứng đúng theo tình huống Trợ lý ảo, hệ thống đề xuất
Tạo tương tác người dùng Dùng để hiển thị thông báo, lỗi, hoặc xử lý hành động của người dùng Popup thông báo, form xác nhận
Kiểm soát luồng chạy Giúp chương trình chạy đúng thứ tự, đúng logic mong muốn Điều khiển máy móc, hệ thống tự động

Minh họa tác dụng của câu lệnh điều kiện

Minh họa:

CÂU LỆNH IF (DẠNG 1)

Định nghĩa:

Câu lệnh if dùng để kiểm tra một điều kiện. Nếu điều kiện đúng (true) thì đoạn mã trong khối if sẽ được thực thi.

Cú pháp:

if (<điều_kiện>) {
    'khối_lệnh';
}

Ví dụ: Kiểm tra số dương

if_example.js
let number = 5;
if (number > 0) {
    console.log("Số dương");
}

Cách hoạt động:

  1. Biến number được khai báo với giá trị 5
  2. Điều kiện number > 0 được kiểm tra (5 > 0 → true)
  3. Vì điều kiện đúng, khối lệnh trong if được thực thi
  4. In ra màn hình: "Số dương"

Thử nghiệm câu lệnh if

Kết quả:

CÂU LỆNH IF...ELSE

Định nghĩa:

Câu lệnh if...else là câu lệnh điều kiện có 2 nhánh:

  • Nếu điều kiện đúng → chạy khối lệnh trong if
  • Nếu điều kiện sai → chạy khối lệnh trong else

Cú pháp:

if (<điều_kiện>) {
    khối_lệnh1;
} else {
    khối_lệnh2;
}

Ví dụ: Kiểm tra đậu/rớt

if_else_example.js
let score = 6.5;
if (score >= 5) {
    console.log("Đậu");
} else {
    console.log("Rớt");
}

Cách hoạt động:

  1. Biến score được khai báo với giá trị 6.5
  2. Điều kiện score >= 5 được kiểm tra (6.5 >= 5 → true)
  3. Vì điều kiện đúng, khối lệnh trong if được thực thi
  4. In ra màn hình: "Đậu"
  5. Khối lệnh trong else bị bỏ qua

Thử nghiệm câu lệnh if...else

Kết quả:

CÂU LỆNH IF...ELSE IF...ELSE

Định nghĩa:

Câu lệnh if...else if...else cho phép kiểm tra nhiều điều kiện khác nhau, theo thứ tự từ trên xuống. Khi gặp điều kiện đúng đầu tiên, chương trình sẽ thực hiện và không kiểm tra tiếp.

Cú pháp:

if (<điều_kiện1>) {
    khối_lệnh1;
} else if (<điều_kiện2>) {
    khối_lệnh2;
} else if (<điều_kiện3>) {
    khối_lệnh3;
} else {
    khối_lệnh_n;
}

Ví dụ: Xếp loại học lực

if_else_if_example.js
let score = 8.5;
if (score >= 9) {
    console.log("Xuất sắc");
} else if (score >= 7) {
    console.log("Khá");
} else if (score >= 5) {
    console.log("Trung bình");
} else {
    console.log("Yếu");
}

Cách hoạt động:

  1. Biến score được khai báo với giá trị 8.5
  2. Kiểm tra điều kiện 1: score >= 9 (8.5 >= 9 → false)
  3. Chuyển sang điều kiện 2: score >= 7 (8.5 >= 7 → true)
  4. Vì điều kiện 2 đúng, thực thi khối lệnh tương ứng
  5. In ra màn hình: "Khá"
  6. Các điều kiện sau và khối else bị bỏ qua

Lưu ý quan trọng:

Thứ tự các điều kiện rất quan trọng! Luôn sắp xếp từ điều kiện khắt khe nhất đến điều kiện dễ nhất. Nếu đặt score >= 5 lên đầu, tất cả số >= 5 sẽ bị xếp loại "Trung bình" dù có thể là "Khá" hoặc "Xuất sắc".

Thử nghiệm câu lệnh if...else if...else

Kết quả:

TOÁN TỬ BA NGÔI (TERNARY OPERATOR)

Định nghĩa:

Toán tử ba ngôi (?:) là cú pháp rút gọn của if...else, dùng khi cần lựa chọn giữa hai giá trị. Nó giúp viết mã ngắn gọn và súc tích hơn, nhất là khi chỉ cần trả về một giá trị.

Cú pháp:

<điều_kiện> ?  :

Ví dụ: Kiểm tra đậu/rớt (dạng rút gọn)

ternary_example.js
let score = 6.5;
let result = (score >= 5) ? "Đậu" : "Rớt";
console.log(result); // "Đậu"

So sánh với if...else:

So sánh cú pháp
// Dùng if...else
let result1;
if (score >= 5) {
    result1 = "Đậu";
} else {
    result1 = "Rớt";
}

// Dùng toán tử ba ngôi (ngắn gọn hơn)
let result2 = (score >= 5) ? "Đậu" : "Rớt";

Khi nào nên dùng toán tử ba ngôi:

  • Khi chỉ có hai lựa chọn đơn giản (đúng/sai)
  • Khi cần gán giá trị cho biến dựa trên điều kiện
  • Khi muốn code ngắn gọn, dễ đọc
  • Không nên dùng khi logic phức tạp hoặc nhiều điều kiện

Thử nghiệm toán tử ba ngôi

Kết quả:

CÂU LỆNH SWITCH...CASE

Định nghĩa:

Câu lệnh switch...case dùng để so sánh một giá trị với nhiều trường hợp khác nhau (case). Nó thường được sử dụng thay cho nhiều câu lệnh if...else if khi so sánh một biến với nhiều giá trị cố định.

Cú pháp:

switch (biểu_thức) {
    case giá_trị_1:
        khối_lệnh_1;
        break;
    case giá_trị_2:
        khối_lệnh_2;
        break;
    ...
    default:
        khối_lệnh_mặc_định;
}

Ví dụ: Hiển thị thứ trong tuần

switch_example.js
let day = 2;

switch (day) {
    case 1:
        console.log("Thứ Hai");
        break;
    case 2:
        console.log("Thứ Ba");
        break;
    case 3:
        console.log("Thứ Tư");
        break;
    case 4:
        console.log("Thứ Năm");
        break;
    case 5:
        console.log("Thứ Sáu");
        break;
    case 6:
        console.log("Thứ Bảy");
        break;
    case 7:
        console.log("Chủ Nhật");
        break;
    default:
        console.log("Không hợp lệ");
}

Quan trọng: Từ khóa break

Từ khóa break là bắt buộc trong hầu hết các trường hợp. Nếu thiếu break, chương trình sẽ tiếp tục thực thi các case tiếp theo cho đến khi gặp break hoặc kết thúc switch (hiện tượng "fall-through").

Khi nào nên dùng switch...case:

  • Khi so sánh một biến với nhiều giá trị cố định
  • Khi có nhiều trường hợp (>= 3) để xử lý
  • Khi các điều kiện là so sánh bằng (===) chứ không phải so sánh phạm vi
  • Code dễ đọc và bảo trì hơn nhiều if...else if

Thử nghiệm câu lệnh switch...case

Kết quả:

TOÁN TỬ LOGIC (&&, ||, !)

Định nghĩa:

Toán tử logic cho phép kết hợp nhiều điều kiện trong một biểu thức, giúp xử lý logic phức tạp hơn.

Toán tử Tên gọi Ý nghĩa Kết quả đúng khi Ví dụ
&& AND (Và) Đúng khi cả hai điều kiện đúng Cả hai đều true (age >= 18 && hasID)
|| OR (Hoặc) Đúng khi ít nhất một điều kiện đúng Ít nhất một true (score >= 8 || bonus >= 2)
! NOT (Phủ định) Phủ định điều kiện (đúng thành sai và ngược lại) Điều kiện false !(age < 18) tương đương age >= 18

Ví dụ: Kiểm tra điều kiện vào cửa

logical_operators.js
let age = 17;
let hasID = true;

if (age >= 18 && hasID) {
    console.log("Được vào cửa");
} else {
    console.log("Không được vào cửa");
}

// Kết hợp nhiều toán tử
let isWeekend = true;
let hasTicket = true;
let hasVaccine = false;

if (hasTicket && (isWeekend || hasVaccine)) {
    console.log("Được tham gia sự kiện");
}

Bảng chân lý (Truth Table):

A B A && B A || B !A
true true true true false
true false false true false
false true false true true
false false false false true

Mẹo sử dụng toán tử logic:

  • Sử dụng dấu ngoặc đơn để làm rõ thứ tự ưu tiên
  • && có độ ưu tiên cao hơn ||
  • Sử dụng ! để đảo ngược điều kiện, giúp code dễ đọc hơn
  • Kết hợp toán tử logic với toán tử so sánh để tạo điều kiện phức tạp

Thử nghiệm toán tử logic

Kết quả:

MẸO VÀ CÚ PHÁP VIẾT NHANH

Truthy và Falsy trong JavaScript:

Trong JavaScript, các giá trị có thể được coi là "truthy" (giống true) hoặc "falsy" (giống false) khi chuyển đổi sang boolean.

Kỹ thuật Cú pháp Ý nghĩa Ví dụ
Truthy check if (a) Kiểm tra nếu a tồn tại (truthy) if (username) ← username không rỗng
Falsy check if (!a) Kiểm tra nếu a không tồn tại (falsy) if (!error) ← không có lỗi
Nullish check a ?? b Trả về a nếu a không null/undefined, ngược lại trả về b let name = input ?? "Khách"
Optional chaining a?.b?.c Truy cập thuộc tính an toàn, tránh lỗi undefined user?.address?.city

Các giá trị Falsy trong JavaScript:

Các giá trị falsy
false // boolean false
0 // số 0
"" // chuỗi rỗng
null // giá trị null
undefined // giá trị undefined
NaN // Not a Number

Ví dụ thực tế:

truthy_falsy_examples.js
// Kiểm tra chuỗi không rỗng
let username = "John";
if (username) {
    console.log("Xin chào " + username);
} else {
    console.log("Vui lòng nhập tên");
}

// Kiểm tra mảng có phần tử
let items = ["apple", "banana"];
if (items.length) { // length > 0 là truthy
    console.log("Có " + items.length + " món hàng");
}

// Gán giá trị mặc định
let userRole = null;
let role = userRole || "guest"; // "guest" vì null là falsy
console.log(role); // "guest"

Mẹo viết code hiệu quả:

  • Sử dụng if (variable) thay vì if (variable !== undefined && variable !== null && variable !== "")
  • Dùng toán tử || để gán giá trị mặc định
  • Sử dụng && để thực thi code có điều kiện: isLogged && showDashboard()
  • Ưu tiên đọc hiểu: đặt điều kiện dương (if (isValid)) thay vì phủ định (if (!isInvalid))

exercise1.js
// Bài 1: Xếp loại học sinh

    console.log("Yếu");
}

Bài 1: Số lẻ chia hết cho 3 nhưng không chia hết cho 5

Nhập vào một số nguyên dương n, kiểm tra xem:

  • Có phải số lẻ
  • Và chia hết cho 3
  • Nhưng không chia hết cho 5

Nếu đúng, in "Thỏa mãn"; ngược lại "Không thỏa mãn"

Bài 2: Tính phụ cấp nhân viên

Dùng hàm prompt nhập:

  • heSoLuong: số thực (VD: 3.5)
  • namCongTac: số nguyên (VD: 12)
  • chucVu: chuỗi "GD" (giám đốc), "TP" (trưởng phòng), "NV" (nhân viên)

Yêu cầu tính phụ cấp:

Phụ cấp chức vụ:

  • GD → 1000
  • TP → 700
  • NV → 500

Điều chỉnh hệ số lương:

  • Nếu namCongTac > 10 → cộng thêm 10% hệ số lương
  • Nếu namCongTac < 5 và chức vụ là "NV" → giảm 5% hệ số lương

Tổng phụ cấp = (phụ cấp chức vụ + hệ số lương sau điều chỉnh) × 100

Bài 3: Tính điểm ưu tiên đại học

Viết chương trình nhập vào:

  • Khu vực: A, B, C, hoặc X (X là không ưu tiên)
  • Đối tượng: 1, 2, 3, hoặc 0 (0 là không ưu tiên)
  • Điểm 3 môn: toán, lý, hóa

Yêu cầu:

Tính tổng điểm gồm:

  1. Tổng 3 môn
  2. Cộng điểm ưu tiên khu vực:
    • A = 2 điểm
    • B = 1 điểm
    • C = 0.5 điểm
    • X = 0 điểm
  3. Cộng điểm ưu tiên đối tượng:
    • 1 = 2.5 điểm
    • 2 = 1.5 điểm
    • 3 = 1 điểm
    • 0 = 0 điểm

Kiểm tra điều kiện:

  • Nếu có môn nào < 1 ⇒ "Rớt vì có điểm liệt"
  • Nếu tổng điểm ≥ 20 ⇒ "Đậu"
  • Ngược lại ⇒ "Rớt"

Bài 4: Tính tiền điện theo bậc thang

Viết chương trình nhập vào số điện sử dụng trong tháng (kWh) và loại khách hàng (sinh hoạt, kinh doanh, sản xuất) và tính tiền điện theo bảng sau:

Bảng giá điện:

Loại khách hàng Đơn giá (VNĐ/kWh)
Sinh hoạt 0–50 kWh: 1800
51–100: 2000
101–200: 2500
>200: 3000
Kinh doanh 3500
Sản xuất 2000

Yêu cầu:

  • Nếu loại là "sinh hoạt" thì tính theo bậc thang (0–50, 51–100, …)
  • Nếu "kinh doanh" hoặc "sản xuất" thì lấy đơn giá * số kWh
  • Nếu nhập sai loại, thông báo "Loại khách hàng không hợp lệ"

Bài 5: Tính tiền gửi tiết kiệm theo kỳ hạn

Ngân hàng cung cấp các gói gửi tiết kiệm với kỳ hạn và lãi suất như sau:

Kỳ hạn Số tháng gửi Lãi suất (%/năm)
Ngắn hạn 3 đến 5 tháng 5%
Trung hạn 6 đến 11 tháng 7%
Dài hạn Từ 12 tháng trở lên 8.5%

Nếu số tháng gửi dưới 3, hệ thống sẽ thông báo là không đủ điều kiện gửi tiết kiệm.

Yêu cầu:

  • Sử dụng hàm prompt để nhập:
    • Số tiền gửi (VND)
    • Số tháng gửi
  • Xác định kỳ hạn dựa trên số tháng gửi.
  • Tính:
    • Tiền lãi = Số tiền gửi × (Lãi suất / 12) × Số tháng
    • Tổng tiền nhận được = Số tiền gửi + Tiền lãi
  • In ra màn hình (console):
    • Kỳ hạn
    • Lãi suất theo kỳ hạn
    • Tiền lãi
    • Tổng tiền nhận được

Bài 5: Tính tiền gửi tiết kiệm theo kỳ hạn

Mô tả bài toán

Tính tiền lãi gửi tiết kiệm theo kỳ hạn:

Ngắn hạn (3-5 tháng): 5%/năm
Trung hạn (6-11 tháng): 7%/năm
Dài hạn (≥12 tháng): 8.5%/năm

Công thức:
Tiền lãi = Số tiền gửi × (Lãi suất / 12) × Số tháng
Tổng tiền = Số tiền gửi + Tiền lãi

Nếu số tháng < 3: "Không đủ điều kiện gửi tiết kiệm"
