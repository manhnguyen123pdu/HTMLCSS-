I. DOM LÀ GÌ?

Định nghĩa DOM

DOM (Document Object Model) là một mô hình đối tượng đại diện cho cấu trúc của tài liệu HTML dưới dạng cây (tree structure). Mỗi phần tử HTML (thẻ, văn bản, thuộc tính, v.v.) được biểu diễn như một đối tượng (object) trong JavaScript mà ta có thể truy cập và thay đổi.

Hiểu nôm na: Khi trang HTML được load, trình duyệt chuyển nó thành một cây DOM để bạn điều khiển bằng JavaScript.

Ví dụ HTML

index.html
<!DOCTYPE html>
<html>
  <head>
    <title>Demo DOM</title>
  </head>
  <body>
    <h1 id="title">Chào bạn!</h1>
    <p class="content">Đây là ví dụ về DOM.</p>
  </body>
</html>

→ Cây DOM tương ứng.

Document
└── html
    ├── head
    │   └── title
    └── body
        ├── h1#title
        └── p.content

II. TRUY CẬP PHẦN TỬ TRONG DOM

1. document.getElementById()

➤ Cú pháp:

document.getElementById()
document.getElementById("id");
  • document: đại diện toàn bộ tài liệu HTML.
  • getElementById("id"): trả về một phần tử duy nhất có id = "id".

➤ Ví dụ:

HTML + JS
<h1 id="myTitle">Xin chào</h1>

let title = document.getElementById("myTitle");
console.log(title.textContent); // Kết quả: "Xin chào"

2. document.getElementsByClassName()

➤ Cú pháp:

document.getElementsByClassName()
document.getElementsByClassName("className");
  • Trả về HTMLCollection gồm các phần tử có cùng class.
  • Không dùng được các phương thức mảng như .forEach(), phải chuyển sang mảng nếu cần.

➤ Ví dụ:

HTML + JS
<p class="note">Ghi chú 1</p>
<p class="note">Ghi chú 2</p>

let notes = document.getElementsByClassName("note");
console.log(notes.length);
console.log(notes[0].textContent);

3. document.getElementsByTagName()

➤ Cú pháp:

document.getElementsByTagName()
document.getElementsByTagName("tagName");
  • Trả về HTMLCollection chứa các phần tử có cùng tên thẻ.

➤ Ví dụ:

HTML + JS
<p>Đoạn 1</p>
<p>Đoạn 2</p>

let paras = document.getElementsByTagName("p");
for (let i = 0; i < paras.length; i++) {
    console.log(paras[i].textContent);
}

4. document.querySelector() và querySelectorAll()

➤ Cú pháp:

querySelector() và querySelectorAll()
document.querySelector("cssSelector");
document.querySelectorAll("cssSelector");
  • querySelector → trả về phần tử đầu tiên khớp với CSS selector.
  • querySelectorAll → trả về NodeList (giống mảng) chứa tất cả phần tử khớp.

➤ Ví dụ:

HTML + JS
<div class="box">Box 1</div>
<div class="box">Box 2</div>

let firstBox = document.querySelector(".box");
console.log(firstBox.textContent); // Box 1

let allBoxes = document.querySelectorAll(".box");
allBoxes.forEach((box) => {
    console.log(box.textContent);
});

Ghi chú quan trọng:

  • getElementById() → chỉ dùng được với ID duy nhất.
  • querySelector() dùng được cho tất cả kiểu chọn (ID, class, tag…).
  • querySelectorAll() hỗ trợ CSS selector như div > p, ul li, a[href], v.v.

III. THAY ĐỔI NỘI DUNG VÀ THUỘC TÍNH DOM

1. Thay đổi nội dung văn bản

element.innerHTML

Định nghĩa: Lấy hoặc gán nội dung HTML bên trong phần tử.

Lưu ý: Có thể chèn thẻ HTML, nên cẩn thận khi lấy từ người dùng (nguy cơ XSS).

➤ Ví dụ:

innerHTML
<div id="content">Xin chào</div>

document.getElementById("content").innerHTML = "<b>Chào bạn!</b>";
// Output HTML:
<div id="content"><b>Chào bạn!</b></div>

element.textContent

Định nghĩa: Lấy hoặc gán nội dung chỉ văn bản, bỏ qua thẻ HTML.

➤ Ví dụ:

textContent
<div id="content">Xin chào</div>

document.getElementById("content").textContent = "<b>Chào bạn!</b>";
// Kết quả hiển thị:
<b>Chào bạn!</b>

So sánh nhanh:

Thuộc tính Gán HTML được? Hiển thị thẻ HTML?
innerHTML ✅ ✅
textContent ❌ ❌

2. Thay đổi thuộc tính (Attribute)

element.setAttribute(name, value)

Đặt hoặc thay đổi giá trị của một thuộc tính HTML.

➤ Ví dụ:

setAttribute
<img id="img1" src="old.jpg" />

document.getElementById("img1").setAttribute("src", "new.jpg");

element.getAttribute(name)

Lấy giá trị của thuộc tính.

➤ Ví dụ:

getAttribute
let src = document.getElementById("img1").getAttribute("src");
console.log(src); // new.jpg

element.removeAttribute(name)

Xóa thuộc tính khỏi phần tử.

➤ Ví dụ:

removeAttribute
document.getElementById("img1").removeAttribute("src");

3. Làm việc với class (CSS Class)

  • element.classList.add("className") → Thêm một class vào phần tử.
  • element.classList.remove("className") → Xóa class khỏi phần tử.
  • element.classList.toggle("className") → Nếu chưa có class → thêm. Nếu đã có → xóa.
  • element.classList.contains("className") → Trả về true nếu phần tử có class đó.

➤ Ví dụ:

classList
let box = document.getElementById("box");

box.classList.add("blue"); // Thêm class "blue"
box.classList.remove("red"); // Xóa class "red"
box.classList.toggle("hidden"); // Nếu có thì xóa, không có thì thêm
console.log(box.classList.contains("blue")); // true

4. Thay đổi thuộc tính đặc biệt (như value, src, href, style)

Thuộc tính Ý nghĩa Ví dụ
value Giá trị trong input input.value = "abc"
href Đường dẫn liên kết a.href = "https://..."
src Nguồn ảnh/video img.src = "image.jpg"
style CSS nội tuyến el.style.color = "red"

➤ Ví dụ thay đổi style:

Thay đổi style
let el = document.getElementById("title");
el.style.fontSize = "24px";
el.style.color = "green";

IV. TẠO, CHÈN, VÀ XÓA PHẦN TỬ DOM

1. Tạo 1 phần tử mới bằng JS

document.createElement("tagName")

Định nghĩa: Tạo một phần tử HTML mới (chưa được hiển thị trên trang). Chỉ tạo trong bộ nhớ, phải chèn vào DOM thì mới thấy.

Cú pháp:

createElement
let element = document.createElement("tag-name");

Ví dụ: Tạo thẻ <p> mới

Tạo phần tử
let p = document.createElement("p");
p.textContent = "Đây là đoạn văn mới";

Sau đoạn này:

  • Đã có 1 phần tử <p> nằm trong bộ nhớ JS.
  • Chưa xuất hiện trên trang HTML.

2. Chèn phần tử vào DOM

parent.appendChild(child)

Định nghĩa: Gắn phần tử con vào cuối cùng trong phần tử cha.

Cú pháp:

appendChild
cha.appendChild(con);

Ví dụ: Chèn đoạn văn vào <div id="box">

Chèn phần tử
let box = document.getElementById("box");
let p = document.createElement("p");
p.textContent = "Nội dung mới";
box.appendChild(p);

Kết quả:

Kết quả HTML
<div id="box">
  <p>Nội dung mới</p>
</div>

3. Xóa phần tử khỏi DOM

element.remove()

Định nghĩa: Xóa chính phần tử đó ra khỏi DOM.

Ví dụ:

remove()
<button onclick="deleteMe(this)">Xóa tôi</button>

function deleteMe(btn) {
    btn.remove();
}

V. SỰ KIỆN TRONG JS

1. SỰ KIỆN LÀ GÌ?

Định nghĩa: Sự kiện (event) trong DOM là các hành động xảy ra trên trang như: người dùng click, gõ phím, di chuột, cuộn trang, thay đổi input, v.v. Bạn có thể “lắng nghe” những sự kiện này và viết mã phản hồi lại khi sự kiện xảy ra.

2. CÁCH GẮN SỰ KIỆN (3 CÁCH)

CÁCH 1 – Gắn trực tiếp vào HTML (ít dùng)

Cách 1
<button onclick="alert('Bạn đã bấm!')">Bấm tôi</button>

CÁCH 2 – Gắn qua thuộc tính on... trong JS

Cách 2
<button id="btn1">Bấm tôi</button>

document.getElementById("btn1").onclick = function () {
    alert("Bạn đã bấm vào nút!");
};

Dễ hiểu, nhưng không thể gắn nhiều sự kiện cùng lúc.

CÁCH 3 – Dùng addEventListener (Khuyên dùng ✅)

Cách 3
<button id="btn2">Click Me</button>

let btn = document.getElementById("btn2");

btn.addEventListener("click", function () {
    alert("Đây là cách đúng chuẩn!");
});

Cho phép gắn nhiều sự kiện cùng lúc và dễ kiểm soát.

3. DANH SÁCH CÁC SỰ KIỆN PHỔ BIẾN

Sự kiện Mô tả
click Khi người dùng click chuột
dblclick Nhấn đúp chuột
mouseover Khi rê chuột vào
mouseout Khi chuột rời khỏi
keydown Khi nhấn phím
keyup Khi thả phím
change Khi input/select thay đổi giá trị
submit Khi form được gửi
input Khi người dùng gõ vào input

4. ĐỐI TƯỢNG event

Khi một sự kiện xảy ra, JS tạo ra một đối tượng event chứa thông tin chi tiết về sự kiện.

Đối tượng event
<button id="btn2">Click Me</button>

let btn = document.getElementById("btn2");

btn.addEventListener("click", function (e) {
    console.log(e); // Đối tượng sự kiện
    console.log(e.target); // Phần tử được click
});

e hoặc event Là đối tượng chứa thông tin như vị trí click, phím bấm, element liên quan...
