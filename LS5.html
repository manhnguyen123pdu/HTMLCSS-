MẢNG LÀ GÌ?

Định nghĩa

Mảng (Array) trong JavaScript là một kiểu dữ liệu đặc biệt dùng để lưu trữ nhiều giá trị trong một biến duy nhất.

Đặc điểm quan trọng

  • Các giá trị trong mảng được lưu theo thứ tự
  • Mỗi giá trị có một chỉ số (index) riêng để truy xuất
  • Index bắt đầu từ 0
  • Mảng có thể chứa nhiều kiểu dữ liệu khác nhau (number, string, boolean, object,...)

Cách hình dung:

Giống như 1 ngăn kéo đựng đồ. Mỗi ngăn là một phần tử trong mảng:

Ví dụ minh họa
let arr = ["Táo", "Cam", "Chuối"]
// Index: 0 1 2
Index 0
"Táo"
Index 1
"Cam"
Index 2
"Chuối"

  • Dễ thao tác hàng loạt: thêm, xoá, sắp xếp, lọc...

Cách 2: Sử dụng từ khoá new (Array constructor)
Cú pháp
let numbers = new Array(1, 2, 3);

Cú pháp
let empty = [];

Ví dụ
let last = fruits[fruits.length - 1];

PHƯƠNG THỨC TRONG MẢNG

5.1 push() – Thêm vào cuối mảng

Cú pháp
array.push(element1, element2, ...)
Tên tham số Ý nghĩa
element1 Phần tử đầu tiên muốn thêm
element2 (Tuỳ chọn) phần tử thứ hai,...

Đặc điểm:

  • Thêm phần tử vào cuối mảng gốc
  • Không tạo mảng mới, thay đổi trực tiếp mảng cũ
  • Trả về độ dài mới của mảng
Ví dụ
let arr = [1, 2, 3];
let newLength = arr.push(4); // thêm 4 vào cuối
console.log(arr); // [1, 2, 3, 4]
console.log(newLength); // 4

5.2 unshift() – Thêm vào đầu mảng

Cú pháp
array.unshift(element1, element2, ...)

Đặc điểm:

  • Thêm phần tử vào đầu mảng gốc
  • Không tạo mảng mới
  • Trả về độ dài mới của mảng
Ví dụ
let arr = [1, 2, 3];
let newLength = arr.unshift(0); // thêm 0 vào đầu
console.log(arr); // [0, 1, 2, 3]
console.log(newLength); // 4

5.3 splice() – Thêm vào vị trí bất kỳ

Cú pháp
array.splice(start, deleteCount, item1, item2, ...)
Tên tham số Ý nghĩa
start Vị trí (index) bắt đầu thao tác (thêm/xoá)
deleteCount Số phần tử muốn xoá kể từ vị trí start
item1, item2… (Tuỳ chọn) các phần tử muốn chèn vào tại vị trí start

Đặc điểm:

  • Có thể thêm ở giữa, đầu, cuối
  • Không tạo mảng mới (thay đổi mảng gốc)
  • Trả về mảng các phần tử bị xóa (nếu có)
Ví dụ
let arr = [1, 2, 4, 5];
arr.splice(2, 0, 3); // tại vị trí 2, xóa 0 phần tử, thêm số 3
console.log(arr); // [1, 2, 3, 4, 5]

5.4 Toán tử spread (...) – Tạo mảng mới

Đặc điểm:

  • Không làm thay đổi mảng gốc
  • Tạo mảng mới với phần tử thêm vào bất kỳ đâu
  • Rất phù hợp trong lập trình hàm (functional programming)
Ví dụ
let arr = [2, 3, 4];
let newArr = [1, ...arr]; // thêm 1 vào đầu, tạo mảng mới
console.log(newArr); // [1, 2, 3, 4]
console.log(arr); // [2, 3, 4] – mảng gốc không đổi

5.5 pop() – Xoá phần tử cuối cùng

Đặc điểm:

  • Không tạo mảng mới
  • Thay đổi mảng gốc
  • Trả về phần tử đã bị xoá
Ví dụ
let arr = [1, 2, 3];
let removed = arr.pop();
console.log(arr); // [1, 2]
console.log(removed); // 3

5.6 shift() – Xoá phần tử đầu tiên

Đặc điểm:

  • Không tạo mảng mới
  • Thay đổi mảng gốc
  • Trả về phần tử đã bị xoá
Ví dụ
let arr = [1, 2, 3];
let removed = arr.shift();
console.log(arr); // [2, 3]
console.log(removed); // 1

Khái niệm

Duyệt mảng (array iteration) là quá trình đi qua từng phần tử trong mảng, thường để:

  • In ra giá trị
  • Tính toán (cộng, lọc, tìm...)
  • Thay đổi hoặc tạo ra mảng

6.1 for thường (có chỉ số)

Đặc điểm:

  • Duyệt từng phần tử bằng chỉ số i
  • Có thể thay đổi phần tử trong mảng
Cú pháp
for (let i = 0; i < arr.length; i++) {
  // sử dụng arr[i]
}
Ví dụ
let arr = [10, 20, 30];
for (let i = 0; i < arr.length; i++) {
  console.log(`Phần tử thứ ${i}: ${arr[i]}`);
}

6.2 for...of (duyệt giá trị)

Đặc điểm:

  • Duyệt trực tiếp qua giá trị từng phần tử
  • Không biết chỉ số
  • Không nên dùng để thay đổi mảng gốc
Cú pháp
for (let value of arr) {
  // sử dụng value
}
Ví dụ
let arr = [10, 20, 30];
for (let value of arr) {
  console.log(value); // 10, 20, 30
}

6.3 for...in (duyệt chỉ số – không khuyến khích với mảng)

Đặc điểm:

  • Duyệt qua các chỉ số (index)
  • Dễ nhầm lẫn với đối tượng (object)
  • Không nên dùng để duyệt mảng trong thực tế
Ví dụ
let arr = ['a', 'b', 'c'];
for (let i in arr) {
  console.log(i, arr[i]);
}

MẢNG LỒNG & DESTRUCTURING

8.1 Mảng lồng (Nested Arrays)

Mảng có thể chứa các mảng khác bên trong.

Ví dụ
let matrix = [
  [1, 2],
  [3, 4]
];
console.log(matrix[1][0]); // 3

8.2 Destructuring (Phân rã mảng)

Lấy giá trị từ mảng và gán vào các biến riêng biệt.

Ví dụ
let [a, b] = [10, 20];
console.log(a); // 10
console.log(b); // 20

8.3 Spread Operator (...)

Toán tử spread cho phép sao chép hoặc kết hợp các mảng.

Ví dụ
let arr1 = [1, 2];
let arr2 = [...arr1, 3];
console.log(arr2); // [1, 2, 3]

GIẢI THUẬT VỚI MẢNG

Bài toán 1: Tính tổng các phần tử trong mảng

Mục tiêu:

Tính tổng tất cả các số trong một mảng.

Tư duy giải thuật:

  1. Khởi tạo biến sum = 0
  2. Duyệt qua từng phần tử trong mảng
  3. Cộng từng phần tử vào biến sum
  4. Sau khi duyệt xong, in hoặc trả về sum
Ví dụ
let arr = [1, 2, 3, 4];
let sum = 0;

for (let i = 0; i < arr.length; i++) {
  sum += arr[i];
}

console.log(sum); // 10

Bài toán 2: Tìm phần tử lớn nhất / nhỏ nhất

Mục tiêu:

Tìm giá trị lớn nhất và nhỏ nhất trong mảng.

Tư duy giải thuật:

  1. Gán max và min bằng phần tử đầu tiên arr[0]
  2. Duyệt qua mảng:
  3. Nếu phần tử lớn hơn max thì cập nhật max
  4. Nếu phần tử nhỏ hơn min thì cập nhật min
Ví dụ
let arr = [3, 7, 2, 9, 4];
let max = arr[0];
let min = arr[0];

for (let i = 1; i < arr.length; i++) {
  if (arr[i] > max) max = arr[i];
  if (arr[i] < min) min=arr[i];
}

console.log("Max:", max); // 9
console.log("Min:", min); // 2

Bài toán 3: Đảo ngược mảng

Mục tiêu:

Tạo một mảng mới chứa các phần tử đảo ngược của mảng ban đầu.

Tư duy giải thuật:

  1. Duyệt từ cuối mảng về đầu
  2. Dùng push() để thêm phần tử vào mảng mới
Ví dụ
let arr = [1, 2, 3, 4];
let reversed = [];

for (let i = arr.length - 1; i >= 0; i--) {
  reversed.push(arr[i]);
}
console.log(reversed); // [4, 3, 2, 1]

Bài toán 4: Loại bỏ phần tử trùng nhau

Mục tiêu:

Lọc ra các phần tử duy nhất, bỏ các phần tử bị lặp.

Tư duy giải thuật:

  1. Tạo mảng mới unique
  2. Với mỗi phần tử trong mảng:
  3. Nếu nó chưa có trong unique thì thêm vào
  4. Dùng includes() để kiểm tra trùng
Ví dụ
let arr = [1, 2, 2, 3, 4, 4, 5];
let unique = [];

for (let i = 0; i < arr.length; i++) {
  if (!unique.includes(arr[i])) {
    unique.push(arr[i]);
  }
}

console.log(unique); // [1, 2, 3, 4, 5]

Bài toán 5: Đếm số lần xuất hiện của phần tử

Mục tiêu:

Biết mỗi phần tử xuất hiện bao nhiêu lần trong mảng.

Tư duy giải thuật:

  1. Tạo một mảng rỗng result để chứa các cặp [value, count]
  2. Duyệt từng phần tử val trong mảng arr
  3. Với mỗi val, kiểm tra xem nó đã có trong result hay chưa
  4. Nếu có, tăng biến đếm (count) của nó lên
  5. Nếu chưa có, thêm [val, 1] vào result
Ví dụ
let arr = [1, 2, 2, 3, 1, 1, 4];
let result = [];

for (let i = 0; i < arr.length; i++) {
  let found = false;

  // Kiểm tra phần tử đã có trong result chưa
  for (let j = 0; j < result.length; j++) {
    if (result[j][0] === arr[i]) {
      result[j][1]++; // tăng count
      found = true;
      break;
    }
  }

  // Nếu chưa có thì thêm mới vào result
  if (!found) {
    result.push([arr[i], 1]);
  }
}

// In kết quả
for (let i = 0; i < result.length; i++) {
  console.log(`Phần tử ${result[i][0]} xuất hiện ${result[i][1]} lần`);
}

Bài toán 6: Sắp xếp tăng dần hoặc giảm dần

Giải thuật Bubble Sort (Sắp xếp nổi bọt)

Ý tưởng:

  1. Lặp qua mảng nhiều lần
  2. Mỗi lần duyệt, nếu phần tử sau < phần tử trước (với tăng dần), thì đổi chỗ
  3. Làm đến khi không còn cặp nào cần đổi nữa
Bubble Sort - Tăng dần
let arr = [5, 2, 8, 1, 3];
for (let i = 0; i < arr.length - 1; i++) {
  for (let j = 0; j < arr.length - 1 - i; j++) {
    if (arr[j] > arr[j + 1]) {
      // hoán đổi
      let temp = arr[j];
      arr[j] = arr[j + 1];
      arr[j + 1] = temp;
    }
  }
}
console.log("Tăng dần:", arr); // [1, 2, 3, 5, 8]

Giảm dần: Chỉ cần đổi dấu điều kiện > thành << /p>

Giải thuật Selection Sort (Sắp xếp chọn)

Ý tưởng:

  1. Duyệt qua từng vị trí trong mảng
  2. Mỗi vị trí, tìm phần tử nhỏ nhất (hoặc lớn nhất) phía sau nó
  3. Đổi chỗ phần tử hiện tại với phần tử nhỏ nhất (hoặc lớn nhất) đó
Selection Sort
let arr = [5, 2, 8, 1, 3];

for (let i = 0; i < arr.length - 1; i++) {
  let minIndex = i;
  for (let j = i + 1; j < arr.length; j++) {
    if (arr[j] < arr[minIndex]) {
      minIndex = j;
    }
  }

  // Hoán đổi
  let temp = arr[i];
  arr[i] = arr[minIndex];
  arr[minIndex] = temp;
}

console.log("Tăng dần:", arr); // [1, 2, 3, 5, 8]

Bài toán 7: Tìm tất cả chỉ số của phần tử X

Mục tiêu:

Tìm tất cả các vị trí (index) mà phần tử X xuất hiện.

Tư duy giải thuật:

  1. Duyệt mảng
  2. Nếu phần tử bằng X → lưu lại chỉ số
Ví dụ
let arr = [1, 2, 3, 2, 4, 2];
let x = 2;
let indexes = [];

for (let i = 0; i < arr.length; i++) {
  if (arr[i] === x) {
    indexes.push(i);
  }
}

console.log(indexes); // [1, 3, 5]

Bài toán 8: Tìm cặp phần tử có tổng bằng X

Mục tiêu:

Tìm tất cả cặp số trong mảng có tổng bằng một số X cho trước.

Tư duy giải thuật:

  1. Duyệt 2 vòng lặp lồng nhau (i, j)
  2. Với mỗi cặp (i, j) khác nhau → kiểm tra tổng
Ví dụ
let arr = [1, 2, 3, 4, 5];
let target = 6;

for (let i = 0; i < arr.length; i++) {
  for (let j = i + 1; j < arr.length; j++) {
    if (arr[i] + arr[j] === target) {
      console.log(`[${arr[i]}, ${arr[j]}]`);
    }
  }
}
// Kết quả: [1,5], [2,4]

Bài toán 9: Gộp nhiều mảng thành một

Mục tiêu:

Tạo mảng mới chứa tất cả phần tử của 2 hay nhiều mảng.

Tư duy giải thuật:

  1. Dùng toán tử spread ... hoặc concat()
Ví dụ
let a = [1, 2];
let b = [3, 4];
let c = [5];

let merged = [...a, ...b, ...c];
console.log(merged); // [1, 2, 3, 4, 5]

Bài 5: Di chuyển số 0 về cuối mảng

Bài 6: Đếm cặp số có tổng bằng K

Bạn được giao nhiệm vụ tổ chức một trò chơi "bắt cặp". Có một danh sách gồm các số nguyên, đại diện cho điểm số của từng người chơi. Một cặp được xem là "hoàn hảo" nếu tổng điểm của họ đúng bằng một số nguyên K đã cho trước.

Lưu ý: Một cặp chỉ tính 1 lần (không phân biệt vị trí), không dùng lại 1 phần tử cho nhiều cặp.

Input arr K Output Giải thích
[1, 5, 3, 7, 9] 10 2 (1,9) và (3,7)
[2, 4, 6, 8, 10] 12 2 (2,10) và (4,8)
[1, 2, 3, 4, 5] 7 2 (2,5) và (3,4)
[0, 1, 2, 3, 4] 4 2 (0,4) và (1,3)
